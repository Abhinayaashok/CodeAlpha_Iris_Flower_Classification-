# Iris Flower Classification using Machine Learning

## 📌 Project Overview
This project classifies Iris flowers into:
- Iris Setosa
- Iris Versicolor
- Iris Virginica

using Machine Learning algorithms.

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## 🤖 Algorithm Used
K-Nearest Neighbors (KNN)

## 📂 Dataset Features
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## 🚀 Steps Performed
1. Import libraries
2. Load dataset
3. Split data
4. Train model
5. Predict results
6. Evaluate accuracy

## 📊 Output
The model predicts flower species with high accuracy.
