# Iris Flower Classification using Machine Learning

## Aim
To classify iris flowers into:
- Iris Setosa
- Iris Versicolor
- Iris Virginica

using Machine Learning techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Machine Learning Algorithm
K-Nearest Neighbors (KNN)

## Dataset Features
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## Steps
1. Import libraries
2. Load dataset
3. Split training and testing data
4. Train the model
5. Predict output
6. Evaluate accuracy

## Output
The model predicts iris flower species with high accuracy.
